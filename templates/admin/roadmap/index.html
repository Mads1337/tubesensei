Database & Models

  • Proxy support for transcript extraction

Key Architectural Decisions

Async Throughout

All database and HTTP operations are non-blocking for maximum throughput and responsiveness.

Agent-Based Pipeline

Modular, reusable discovery agents with event emission for flexible and extensible processing.

Checkpoint/Resume

Campaigns and agents preserve state for pause and resume, ensuring no work is lost.

Cost Tracking

Token usage and LLM cost estimation at every layer for full transparency and budget control.

Multi-Source Discovery

Search, channel expansion, and similar videos all feed the discovery pipeline for comprehensive coverage.

Distributed Processing

Celery workers with dedicated queues for different workload types enable scalable, parallel processing.
